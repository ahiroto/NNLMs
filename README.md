# NNLMs: Neural Network Language Models

## requirements
* python  (>=2.7.6)
* chainer (>=1.17.0)
* numpy   (>=1.11)
* Six     (>=1.9)

## preparation

Install Chainer via PyPl:
```bash
$ pip install chainer
```

If you want to get more details for the installation of chainer,
see `https://github.com/pfnet/chainer` .
